# Lar dos Peludos

Site institucional de adoção de animais — projeto educacional.

## Execução local
- Abra `index.html` ou use Live Server.
- Para gravação real do formulário: configurar Supabase e deploy no Vercel (ver instruções).

## Estrutura
- pages/: about, adopt, tips, contact, login
- assets/: css, js, images (20 imagens: pet_1.jpg .. pet_20.jpg)
- api/submit.js -> função serverless (Vercel)
